# ADR-108 — Catalogação idempotente após promoção

Somente outputs promovidos e aprovados tornam-se pesquisáveis. Finals e proxies viram assets; mídia bruta sintética reaproveitável vira segmento. Temporários, falhas e rejeições são ignorados. A chave `workspace + artifact + manifest` torna o reprocessamento convergente, enquanto rights, consent, parents e geração são herdados integralmente no lineage.
