# ADR-131 — Capture protocol, diagnostics and multicamera direction

The product explains synchronization prerequisites before upload for teacher+screen, podcast, react and multicamera capture. A decodable flash+chirp marker may provide independent evidence; otherwise diagnostics expose confidence, residual, drift, coverage and editable manual anchors. Automatic editing remains blocked below the evidence threshold.

Multicamera direction selects only covered angles using speaker, demonstration, reaction, quality, continuity and protected choices. Camera matching occurs before creative LUTs. Color criticism compares technical and output states while honoring declared creative intent.
