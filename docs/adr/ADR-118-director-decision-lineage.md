# ADR-118 — Decision log ligado ao frame final

Cada decisão persiste alternativas, evidência, confidence, score, custo e actor. IDs de run, plan node, command e artifact formam a cadeia auditável; o RenderElementMap fecha a relação até ranges de frames finais. A API permite registrar e consultar, enquanto o painel mostra resumo compacto e detalhe expansível sem esconder alternativas rejeitadas.
