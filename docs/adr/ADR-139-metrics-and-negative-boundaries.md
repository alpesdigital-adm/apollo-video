# ADR-139 — Metrics and negative product boundaries

Metrics use a bounded dictionary, safe dimensions and source-channel attribution. Time saved is an explicitly non-causal historical estimate. Quality baselines and regression windows are versioned.

The twelve non-goals are executable denial policies: no general NLE, false sync precision, fabricated claims, degraded cleanup, Cartesian generation, uncriticized provider output, rights bypass, spectacle by default, prompt injection through content, causal performance claims, direct v1 coupling or public internal resources.
