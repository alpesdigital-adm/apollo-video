# ADR-114 — StoryPlan antes do EditPlan

Atos e blocos narrativos definem papel, intenção, dependências, candidatos de source e duração alvo antes de qualquer corte. Cold open referencia um range posterior sem duplicá-lo. Claims, qualifiers, prova e CTA permanecem ligados ao bloco. Somente uma história com cobertura integral, dependências válidas e duração possível pode compilar para EditPlan.
