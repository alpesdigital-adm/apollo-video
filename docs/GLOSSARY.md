# Canonical domain glossary

- **Master**: immutable verified original bytes. **Derivative**: versioned output with recipe and lineage.
- **MediaSegment / SpeechSegment / EvidenceSegment**: virtual source range; spoken complete thought; claim-bearing contextual evidence.
- **TreatmentPlan / StoryPlan / EditPlan**: editorial policy; narrative structure; renderable timeline.
- **FormatVariantPlan / LocalizationVariant / VariantRecipe**: format overlay; locale-owned production; bounded source combination.
- **DirectorRun / QualityReport**: one audited decision run; critic evidence for a specific artifact.
- **ProviderAdapter / ProviderJob**: vendor-neutral capability boundary; normalized durable provider execution.
- **CaptureSession / SyncDiagnostic**: related capture tracks on a canonical clock; synchronization evidence and actions.
- **ApiClient / PublicCapability / PublicOperation / PublicEvent**: external identity; stable action; durable execution; versioned notification.

Unqualified `Scene`, `Asset`, `Version`, `Variant` and `Job` are forbidden in v2 declarations because they hide ownership and lifecycle. Use the qualified concepts above.
