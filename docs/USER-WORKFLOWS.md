# User workflow concepts

**Apollo Sync Marker** provides a decodable flash/chirp for cameras, screens and recorders. Without common evidence, the system requests a manual anchor.

**Cold open** may move an approved later range to the beginning while preserving attribution and narrative meaning. **Voiceover** uses canonical audio timing with B-roll and captions without requiring an on-screen person. **Synthetic Presenter** splits approved audio/script into reusable raw blocks, applies consent/disclosure and critic gates, then composes them like other sources.

Every workflow shows prerequisites, known limitations, costs and fallbacks before paid or irreversible work.
